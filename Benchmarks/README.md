The benchmarks from this directory are used to evaluate the performance of the mySAT solver. They include a variety of CNF formulas that test different aspects of the solver's capabilities, from SAT and UNSAT instances to more complex formulas.

These files was taken from the SATLIB - Benchmark Problems site, where you can these exact benchmarks and more: https://www.cs.ubc.ca/~hoos/SATLIB/benchm.html
